# Zeyvro Turnstile

**Cloudflare Turnstile anti-spam for PrestaShop contact forms**

PrestaShop 8 module by [Zeyvro](https://zeyvro.com).

## Installation

Download the ZIP from the [releases page](https://github.com/zeyvro/zeyvro-prestashop-turnstile/releases) and install via PrestaShop backoffice → Modules → Upload a module.

## Requirements

- PrestaShop 8.x
- PHP 8.1+
